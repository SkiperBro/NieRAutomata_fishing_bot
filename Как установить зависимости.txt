## Установка зависимостей для работы бота

Для работы бота необходимо установить следующие компоненты:

1. Python 3.10 или новее (скачать с python.org)
2. Зависимости Python: pyautogui, soundcard, numpy

### Способ 1: Ручная установка через командную строку
Откройте командную строку (Win + R, введите cmd) и выполните команду:

pip install pyautogui soundcard numpy

### Способ 2: Автоматическая установка
Запустите файл install_dependencies.bat - он автоматически установит все необходимые библиотеки.

### Проверка установки
После установки запустите бот командой:
python nierfishingbot.py

Если возникнут ошибки, убедитесь что Python установлен и добавлен в PATH.
